# My dotfiles

Feel free to use it but at your own risk.

# Install

1) Clone with Git

```
git clone https://github.com/glbmaximilianotulian/dotfiles ~/.dotfiles
```

2) Make install script executable
```
sudo chmod -x ~/.dotfiles/install.sh
```
3) Execute installation script
``` 
~/.dotfiles/install.sh
```
